C++20语言特性

协程

协程（coroutine）是一种特殊的函数，可以暂停和恢复运行。要定义协程，函数体中必须有 co_return、co_wait 或 co_yield 关键字。C++20的协程是无栈协程，除非编译器进行优化，否则其状态将在堆上分配。

  • Nested requirements - 使用 requires 关键字，指定附加条件（就像本地参数那样）。

lambda表达式模板语法

在lambda表达式中使用熟悉的模板语法。

范围循环的初始化器

此功能简化了常见的代码模式，让代码更紧凑，并对常见的生命周期问题提供了一个优雅的解决方案。

[[likely]] 与 [[unlikely]] attributes

向优化器提示被标记的语句有高概率执行。

如果if右边出现了 likely/unlikely 的 attribute，表示分支很可能/不太可能被执行。

也可以用于迭代语句。

隐式捕获this已过时

在lambda表达式中使用 [=] 隐式捕获this指针已经过时，更推荐的做法是显式使用 [=, this] 或 [=, *this]。

非类型模板使用类类型

类现在可以用在非参数模板里了。模板参数中传入的对象具有型别 const T，其中T是这个对象的类型，并具有静态存储持续期。

constexpr虚函数

虚函数现在能添加 constexpr 并在编译期评估。constexpr的虚函数可以重写非constexpr虚函数，反之亦然。

explicit(bool)

可以在编译器指定构造函数是否为显式。 explicit(true) 和直接指定 explicit 效果一样。

立即执行函数

类似于 constexpr 函数，但是函数由 consteval 指定，并产生一个常量。这被称为立即执行函数（immediate functions）。

using enum

指定枚举的命名空间来提升可读性。

C++20库特性

概念库

标准库也提供概念库，来创建更多复杂概念。其中包括：

Core language concepts：

  • same_as - 指定两个类型相同。
  • derived_from - 指定一个类型继承自另一个类型。
  • convertible_to - 指定一个类型可以隐式转换为另一个类型。
  • common_with - 指定两个类型共享同一个类型。
  • integral - 指定整型。
  • default_constructible - 指定这个类型的对象可以被默认构造。

Comparison concepts：

  • boolean - 指定这个类型可以使用布尔上下文。
  • equality_comparable - 指定 operator== 可以表示相等关系。

Object concepts：

  • movable - 指定对象可以被移动和交换。
  • copyable - 指定对象可以被拷贝、移动和交换。
  • semiregular - 指定对象可以被拷贝、移动、交换和默认构造。
  • regular - 指定对象既 semiregular 又 equality_comparable。

Callable concepts：

  • invocable - 指定这个可调用的类型可以被指定参数列表调用。
  • predicate - 指定调用类型有布尔返回值。

另见：概念。

同步缓冲输出流

缓冲输出流以确保同步（即输出没有交错）。

std::span

span 是一个对容器查看的视图（非拥有），并提供对连续元素组的边界检查。由于视图并不持有元素，所以构造与拷贝的成本很低。思考这些视图的简单方式就是把它们看作是对数据的引用。与维护指针/迭代器长度不同的是，span 把这些都封装到一个对象里。

span 可以是动态大小的，也可以是固定大小的。固定大小的 span 可以受益于边界检查。

span 不支持 const，所以需要构造只读的 span 时请使用 std::span。

示例：使用动态大小的 span 从不同容器中打印整数。

示例：静态大小的 span 塞进去大小不一致的容器就会编译失败。

位操作

C++20 提供了新的 头文件并提供了一些位操作，包括 popcount（算二进制有几个1）。

数学常数

数学常数，包括π、e等，在 头文件中。

std::is_constant_evaluated

谓词函数，如果编译期上下文调用就返回 true。

std::make_shared supports arrays

starts_with and ends_with on strings

string（和string_view）现在有 starts_with 和 ends_with 两个成员函数，以检查是否以给定字符串开始或结束。

检测关联容器是否包含元素

现在关联容器，例如 set 或者 map，具有 contains 成员函数，以代替“find 并且与迭代器末尾比较”的编程方式。

std::bit_cast

一个更安全的方式，把一个对象从一个类型转换到另一个类型。

std::midpoint

更安全地计算两个整数的中点（无溢出）。

std::to_array

把数组或者类似数组的对象转换为 std::array。
