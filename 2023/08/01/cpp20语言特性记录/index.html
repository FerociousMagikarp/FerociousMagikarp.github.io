C++20语言特性

协程

协程（coroutine）是一种特殊的函数，可以暂停和恢复运行。要定义协程，函数体中必须有 co_return、co_wait 或 co_yield 关键字。C++20的协程是无栈协程，除非编译器进行优化，否则其状态将在堆上分配。

上述 range 生成器函数从 start 到 end（不包含）生成值，每个迭代步骤产生的当前值存储在 start 中。生成器在每次 range 函数调用时都保持其状态（这个用例中，调用就是for循环中的每次迭代）。co_yield 接受给定表达式，产生其值，并在此节点挂起协程。协程恢复后，将在 co_yield 节点之后继续执行。
